REST <---> Tarantool exchange protocol
Пакеты запросов к Tarantool имееют общий вид и состоят из трёх элементов:
1. 4 байта — размер msgpack.
2. 1 байт — тип операции.
3. Msgpack соответствующего размера с информацией.

Для операции PUT msgpack состоит из массива, содержащего user_id и 
последовательность индивидов для записи (create, update): 
[user_id, individual_1, ..., individual_n].

Ответ на запрос PUT представляется masgpack'ом из массива, который содержит
общий код выполнения операции и коды результатов операции вставки: 
[general_code, code_1, ..., code_n].

Для операции GET msgpack состоит из массива, содержащего user_id и 
последовательность uri индивидов для чтения: 
[user_id, uri_1, ..., uri_n].

Ответ на запрос GET представляется msgpack'ом из массива, который содержит
общий код выполнения операции, далее идёт последовательность кодов результатов
для чтения, если код чтения соответствует успеху (1), то после кода идёт
msgpack индивида, если код соответствует провалу (0), то после него идёт код
следующей операции чтения.
