API по работе с онтологией:

1) Resource[] getDomainProperties (Resource class)
Функция возвращает массив свойств указанного класса, включая все свойства его суперклассов.

2) Resource[] getIndividuals (Resource class)
Функция возвращает массив всех индивидов класса.

3) Resorce[ Resource ] getIndividualProperties (Resource individual)
Функция возвращает ассоциативный массив свойств указанного индивида (object [predicate] getIndividual (subject) ).
